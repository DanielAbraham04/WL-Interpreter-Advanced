Zip file contains all the files required for the project.
Zip file contains a sample input file. Update the below line in main.cpp and pass the input file as required to test the project.
    invoker.loadInstructions("/Users/danielabraham/CLionProjects/mp4/input.wl");

Instruction is a super class and classes Varint, Varlist, Combine, Add, Chs, Copy, Get, Set, Ifs, Gotos, Hlt are subclasses of Instruction.
Node and LinkedList classes contain the implementation of singly linked lists.
Invoker class contains the implementation of the WL Interpreter.
